О проекте

Наш проект — это интерактивный генератор школьных расписаний. С его помощью можно легко создать расписание для классов, учитывая предметы, учителей и другие параметры.
Генератор автоматически рассчитывает оптимальные временные слоты для предметов, минимизируя конфликты. Проект разработан с учётом удобства как для учителей, так и для школьной администрации.

Функциональность

  • Автоматическая генерация расписаний для классов
  • Помощь в ответе на вопросы по поводу норм, установленных Министерством образования Республики Беларусь
  • Помощь в ответе на вопросы по предметной области составления расписания
  • Предоставляет составленное расписание в понятном пользователю виде

Перспективы

В перспективах развития проекта мы планируем:

  • расширить базу знаний
  • создать собственный алгоритм для генерации расписания
  • добавить разные виды учебных заведений
